# RentTool — веб-сервис аренды инструментов

Веб-сервис для аренды инструментов. Позволяет пользователям просматривать доступные инструменты, оформлять аренду и управлять заказами.

---

## Описание

Данный проект представляет собой backend веб-сервис для системы аренды инструментов.

Сервис предоставляет REST API для:

* работы с пользователями
* управления инструментами
* оформления аренды
* хранения данных в PostgreSQL

Проект использовался как практический пример построения production-like backend системы с контейнеризацией и изолированным окружением.

---

## Основная функциональность

* регистрация и авторизация пользователей
* просмотр списка доступных инструментов
* бронирование инструмента
* история аренд
* управление инструментами (CRUD)

---

## Технологии

* Java 17
* Spring Boot
* Spring Web
* Spring Data JPA
* PostgreSQL
* Docker
* Docker Compose

---

## Запуск проекта

Запуск всей системы одной командой:

docker compose up --build
```

После запуска API доступно по адресу:

```
http://localhost:8080
```
