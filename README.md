Веб-сервис для аренды инструментов. Позволяет пользователям просматривать доступные инструменты, оформлять аренду и управлять заказами.

# Описание

Данный проект представляет собой backend веб-сервис для системы аренды инструментов.  
Сервис предоставляет REST API для работы с пользователями, инструментами и арендой, а также хранит данные в базе PostgreSQL.

Проект предназначен для учебных целей и демонстрации работы с Java, Spring и реляционной базой данных.

# Функциональность

- Регистрация и авторизация пользователей
- Просмотр списка доступных инструментов
- Бронирование инструмента
- Просмотр истории аренд
- Управление инструментами (добавление, редактирование, удаление)

# Технологии

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
